# Monaco’s population overview

Dashboard ShinyApps prezentujący dane o wzroście, masie ciała i BMI mieszkańców Monako (oraz innych krajów) w latach 1880 i 1980.

## Struktura dashboardu

- **Introduction** – opis Monako, tabela średniego BMI, surowe dane populacji oraz wnioski dla Monako.
- **Statistics** – statystyki opisowe, boxploty wzrostu i wagi oraz zmiany BMI kobiet i mężczyzn w czasie.
- **World Dimensions** – porównanie rozkładów wzrostu, wagi i BMI między krajami, interaktywne wykresy i tabela kwantyli.
- **Health on Horizon** – „dumbbell plots” zmian mediany wzrostu/wagi w krajach, interpretacje trendów zdrowotnych.
